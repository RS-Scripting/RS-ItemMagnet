# Pause and Resume

RS-ItemMagnet machines can be temporarily paused without removing the machine or changing its configuration.

---

# Why Pause a Machine?

Pausing allows you to temporarily stop item collection while keeping:

* Radius settings
* Filters
* Ownership
* Container assignments

fully intact.

---

# Pausing a Machine

Open the machine menu.

Select:

```text
Pause
```

The machine will immediately stop collecting items.

Any items within range will remain on the ground.

---

# Resuming a Machine

Open the machine menu.

Select:

```text
Resume
```

The machine will resume normal operation and continue collecting items.

---

# Typical Uses

## Maintenance

Pause the machine while modifying nearby farms or storage systems.

## Storage Changes

Pause collection while reorganizing containers.

## Troubleshooting

Pause a machine to isolate automation issues during testing.

---

# Notes

Pausing does not remove or alter machine settings.

The machine can be resumed at any time.
